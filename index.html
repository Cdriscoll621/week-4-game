Pick Your Crystal!

Rules of The Game

1. You will be given a random number from the computer.

2. Start by choosing a crystal from below. Each crystal has a set ammount of points.

3. Win the game by matching you number, by choosing gems, with the random number that has been generated

4. If your number goes over the random number given, you lose.

5. When you restart the game, the value of each gem changes.

Good Luck!
